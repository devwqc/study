# 들어가며

## 폴리필(polyfill)과 트랜스파일(transpile)

최신 기능을 **구버전의 실행 환경에서 동작**할 수 있게 바꿔주는 역할

**폴리필**

- 브라우저가 **지원하지 않는 코드를 브라우저에서 사용**할 수 있도록 변환한 코드 조각이나 플러그인

**트랜스파일**

- 최신 버전의 코드를 **예전 버전의 코드로 변환**하는 과정

## 자바스크립트의 한계

### 동적 타입 언어

자바스크립트는 코드가 실행되는 런타임에 변숫값이 할당될 때 해당 **값의 타입에 따라 변수 타입이 결정**된다.

따라서 의도치 않은 **에러가 발생할 가능성이 높다**.

## 타입스크립트

자바스크립트의 슈퍼셋 언어로 **정적 타이핑**을 제공한다. 컴파일 단계에서 타입 검사를 해주기 때문에 런타임 **에러를 사전에 방지**할 수 있다.

**슈퍼셋(Superset)**

- 기존 언어에 **새로운 기능과 문법을 추가해서 보완하거나 향상**하는 것을 말한다. 슈퍼셋 언어는 **기존 언어와 호환**되며 일반적으로 **컴파일러 등으로 기존 언어 코드로 변환**되어 실행된다.
